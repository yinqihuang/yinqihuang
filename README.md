<h3> Virtual Reality • Perception and Action • Control Theory </h3>

My overarching goal is to understand **human cognition** in **naturalistic environments** using **computational models**. I focus on **sensorimotor integration**, examining how the brain generates predictions, detects prediction errors, and uses these errors to update behaviour.
